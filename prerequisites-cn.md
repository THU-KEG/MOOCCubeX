## prerequisites/cs.json

计算机领域概念先后修的标注与预测。

| field | description |
| ----- | ----------- |
| c1 | 先修概念 |
| c2 | 后修概念 |
| ground_truth | 是否有先后修关系，`1`为有，`0`为无 |
| text_predict | 使用文本特征得到的预测结果 |
| graph_predict | 使用graph特征得到的预测置信度 |
